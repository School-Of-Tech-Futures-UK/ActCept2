## ActCept2

```
git clone git@github.com:School-Of-Tech-Futures-UK/ActCept2.git
```

## Make a change:

```
git add .
git commit -m "Added a new file called my name"
git push
```

## Pull request in the morning:

```
git pull
```

## Install node modules

```
cd actcept
npm install
npm run build
```

## Start the app

```
cd actcept
npm start
```
